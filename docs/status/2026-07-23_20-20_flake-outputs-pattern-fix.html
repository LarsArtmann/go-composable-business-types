Root Cause

The outputs function in flake.nix used a destructuring pattern inputs@{ self, flake-parts, systems, treefmt-nix } without an ellipsis (...). Nix passes all declared inputs to outputs, including nixpkgs. Because nixpkgs was not listed and there was no ... catch-all, Nix rejected it: function 'outputs' called with unexpected argument 'nixpkgs'.

This single missing ... blocked every Nix check — nix-fmt, nix-build-verify, and nix-hash-fix — before they could even start building.
